Example Query

If for instantance you wanted to find out which countries and players failed to score some of their penalty shots you could fill out a query like below.


                    


                        SELECT player_name AS "Player", country_name AS "Country", penalty_shootout.match_no AS "Match", penalty_shootout.score_goal AS "Failed shots" 

                        FROM player_mast, penalty_shootout, soccer_country 

                        WHERE player_mast.player_id = penalty_shootout.player_id 

                        AND soccer_country.country_id = player_mast.team_id 

                        AND penalty_shootout.score_goal = 'N' 

                        ORDER BY penalty_shootout.match_no, player_mast.player_id;

The query above can be broken down like this.

  • The first line chooses the columns we want to show using the SELECT keyword, and how we want them named with the AS keyword. With one being chosen from a database.
  • The second line shows which databases we are pulling our information from.
  • The WHERE keyword show that the following are conditionals. The first conditional joins the players and the penalty tables.
  • In this line the AND keyword at the start shows that the query has to also follow the conditional on this line. This one links the country and player tables together.
